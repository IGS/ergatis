Papyrus



Description
Papyrus is a package of data management scripts and associated workflows. As the names of the command-line tools suggest, these scripts and workflows are primarily concerned with running various types of comparative analyses. Papyrus operates on BSML-formatted data files, and relies on Prism to generate its BSML input files (typically from a chado database) and load its BSML output files back into a chado database. Papyrus also relies on the TIGR Workflow and HTC systems to manage and distribute the execution of its analysis tasks and subtasks.

Papyrus install
The Papyrus package contains data management scripts and associated workflows. The Papyrus package is installed using the standard Perl module install syntax. In addition, you must specify a SCHEMA_DOCS_DIR where the BSML schemas are installed. The WORKFLOW_DOCS_DIR specifies where the workflow template and default configuration files will be stored. 

  cvs co papyrus_install
  cd papyrus_install
  perl Makefile.PL PREFIX=[install_dir] SCHEMA_DOCS_DIR=[schema_docs_dir] WORKFLOW_DOCS_DIR=[workflow_docs_dir]
  make 
  make install
